# KRas-G12D-SOS1
This document contains the information related to building a KRas·SOS1·KRas ternary complex for molecular dynamic simulations. There are eight folders:
1. The KRasGTP_GEFs_BI3406 folder includes:
   1.1. the constructed inintial structure of the wild type BI-3406·(KRasGTP·Mg2+)·SOS1·(KRasGTP·Mg2+) quaternary complex.
   1.2. the input files for molecular dynamic (MD) simulation.
   1.3. all kinds of RMSD values evolving with the simulation time (figure1b). 
   1.4. the distance from the interactional network of mg2+ ion (figure1d). 
   1.5. the distance between D12 of KRas and the around charged residues (figure 2).
   1.6. the pairwise distance (figure 3a).
   1.7. the distance of the important hydrogen bonds (figure 3d). 
   1.8. the distance of the important salt bridges (figure 4).
   1.9. the representative structure for showing the interactions (figure 3b, figure 3c, figure6) and calculating the electrostatic potential energy surface (figure 8).
   1.10. the results from MM/GBSA calculation (figure 7). 
     
2. The KRasGTP_GEFs folder includes:
   2.1. the constructed inintial structure of the wild type (KRasGTP·Mg2+)·SOS1·(KRasGTP·Mg2+) ternary complex.
   2.2. the input files for molecular dynamic (MD) simulation.
   
3. The KRasGTP_G12D_GEFs_BI3406 folder includes:
   3.1. the constructed inintial structure of KRas G12D mutated BI-3406·(KRasGTP·Mg2+)·SOS1·(KRasGTP·Mg2+) quaternary complex.
   3.2. the input files for molecular dynamic (MD) simulation.
   
4. The KRasGTP_G12D_GEFs folder includes:
   4.1. the constructed inintial structure of KRas G12D mutated (KRasGTP·Mg2+)·SOS1·(KRasGTP·Mg2+) ternary complex.
   4.2. the input files for molecular dynamic (MD) simulation.

   
5. The KRasGDP_GEFs_BI3406 folder includes:
   5.1. the constructed inintial structure of  the wild type BI-3406·(KRasGDP·Mg2+)·SOS1·(KRasGTP·Mg2+) quaternary complex.
   5.2. the input files for molecular dynamic (MD) simulation.

   
6. The KRasGDP_GEFs folder includes:
   6.1. the constructed inintial structure of  the wild type (KRasGDP·Mg2+)·SOS1·(KRasGTP·Mg2+) ternary complex.
   6.2. the input files for molecular dynamic (MD) simulation.

   
7. The KRasGDP_G12D_GEFs_BI3406 folder includes:
   7.1. the constructed inintial structure of KRas G12D mutated BI-3406·(KRasGDP·Mg2+)·SOS1·(KRasGTP·Mg2+) quaternary complex.
   7.2. the input files for molecular dynamic (MD) simulation.


    
8. The KRasGDP_G12D_GEFs folder includes:
   8.1. The the constructed inintial structure of KRas G12D mutated (KRasGDP·Mg2+)·SOS1·(KRasGTP·Mg2+) ternary complex.
   8.2. the input files for molecular dynamic (MD) simulation.
